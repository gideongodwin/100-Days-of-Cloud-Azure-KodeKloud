## Day 12: Add and Manage Tags for Azure Virtual Machines

## Task Details:

The Nautilus DevOps team is migrating a portion of their infrastructure to Azure. During the migration, they have created several virtual machines (VMs) in different regions. The team has identified one VM that is not tagged properly so they decided to tag it as needed.

- Add the tag `Environment=dev` to the virtual machine named `datacenter-vm`

## Steps:

1. Sign in to the [Azure Portal](https://portal.azure.com/)

2. On the virtual machines dashboard, select the available vm `datacenter-vm`

    <img width="682" height="330" alt="539039809-57c83464-78b2-454b-9f8b-c1499e656cdb" src="https://github.com/user-attachments/assets/0663bf1b-baaa-4c2d-9a03-1d6970d06f5c" />

 
3. On the `datacenter-vm` overview page, select Add tag.

    <img width="875" height="515" alt="539057752-5bc5b821-0a21-41f2-81ba-44f299ac1366" src="https://github.com/user-attachments/assets/43a1af76-a058-44a5-9f0c-69b1cf7b46da" />

4. Enter a Name (key) and Value for the tag, and then select Apply.
